Requirements
==============
	* Java Run Time Environment 1.3 or Higer

How To Run Application
==========================
Server
======
  * MULTICHAT - used to store to Username, Machine Address and IP Address 
  * run chatserver.jar 

Client
======
 * run chatclient.jar and click on login button
* enter the username and give the machine name of chatserver (if the server runs on same machine just give localhost as a servername) and click Ok Button.
